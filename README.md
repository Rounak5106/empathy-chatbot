# 💙 Empathy Chatbot

An AI-powered Empathy Chatbot designed to create supportive and emotionally aware conversations using Natural Language Processing and AI techniques.
The chatbot aims to understand user emotions and provide context-sensitive responses that feel more human and meaningful.

---
## 🌟 Features
✨ Emotion-aware conversations  
✨ Real-time chatbot interaction  
✨ Context-sensitive empathetic responses  
✨ AI-powered text understanding  
✨ Lightweight and easy deployment  
✨ User-friendly interface

## ⚙️ Installation

Clone the repository:

git clone https://github.com/Rounak5106/empathy-chatbot.git

Move into the project directory:

cd empathy-chatbot

Install required dependencies:

pip install -r requirements.txt

---

## ▶️ Run the Project

Run the application using:

```bash
python app.py
```
If using Streamlit:
```bash
streamlit run app.py
```
The application will start locally.
